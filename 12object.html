자바스크립트 객체

개요

자바스크립트는 기본 자료형을 제외한 나머지는 모두 객체임 자바스크립트는 기본적으로 key와 value로 구성된 속성들의 집합

자바스크립트크의 함수는 1급 시민이므로 값으로 취급 가능 따라서 객체는 데이터를 의미하는 속성과 데이터를 참조하고 조작하는 동작을 의미하는 메서드(함수)로 구성할수 있음

속성property

키와 값으로 구성된 객체의 주된 구성요소

객체 속성 접근하기

객체 속성에 접근하거나 다룰려면 `객체명.키이름` 형식 사용 또는 `객체명[키이름]`으로도 사용가능

객체 내용 순회

for -in문을 사용하면 객체에 포함된 모든 속성에 접근할수 있음

다양한 형태의 객체

객체의 요소로 객체 또는 함수를 포함시킬수 있음

set 객체

배열과 유사한 자료구조 하지만 배열은저장순서를 기억하고 ,중복갓허용하지만 set은 중복값은 허용안함

map

키와 값으로 구성한는 object 와 유사한 자료구조 하지만Object는 속성과 함수를 포함할수 있지만 map 은 속성만 포함할수 있음 --데이털를 빨리찾기위한 자료구조이다
