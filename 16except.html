자바스크립트예외처리except

프로그램의 실행을 중단시키는 요인을 에러 또는 예외라고 함

error:개발자가 조치할 수 없는 시스템상의 오류 ex)네트워크오류, 전원차단의 오류 except(예외);개발자가 조치할수 있는 문법오류 논리오류 실행오류

자바스크립트의 요류는 try-catch문을 통해 관리가능

try { 예외가 발생할만한 코드들 } catch(객체) { 예외처리코드 }

오류 정보 확인

error객체의 name,message,stack 속성으로 확인 가능

유형별 에외처리

오류 유형별로 메세지 처리를 상이하게 하고 싶으면 instanceof 연산자를 사용하면 됨

referenceError:변수선언과 관련되 ㄴ오류

rangeError:범위와 관련된

syntaxError;문볍과 관련된

typeError:변수나 객체의 자료형과 관련된

인위적으로 오류발생시키기

특정 코드에 의해 발생되는 오류 외에도 로직이 정상적으로 수행되는 특정조건을 만족하지 못했을때 의도적으로 오류를 발생시켜 catch 블럭에 의해 예외처리를 해야할 경우가 있음:

이때 'throw 오류객체명' 구문으로 오류를 발생시킬수 있음
