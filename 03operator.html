자바스크립트 연산자

표현식

표현식expression : 리터럴(값), 식별자(변수), 연산자, 함수 호출등의 조합을 의미 표현식은 평가evaluate(표현식이 실행되어 결과값 생성)되어 하나의 값을 만듦

연산자

하나 이상의 표현식을 대상으로 산술,할당,비교,논리,타입 연산을 수행해서 하나의 값을 만드는 기호를 의미 할당 연산자 : 오른쪽 피연산자를 왼쪽 피연산자에 대입하는 기호, 보통 '='를 의미

산술 연산자 : 덧셈,뺄쎔,곱셈,나눗셈 연산을 수행하는 기호. 그외에도 나머지 연산자(%), 증감/감소 연산자(++,--), 거듭제곱(**)등이 있음 산술 연산자 중 + 는 피연산자 중 하나가 문자열인 경우 문자열을 연결하는 기능을 수행함

산술연산자중 +

복합할당연산자 : 할당 연산자와 산술 연산자를 혼합해서 표현식을 간단하게 작성하는데 사용

비교연산자

좌항과 우항의 피연산자를 비교해서 큰지 작은지 같은지 알아보는 연산자 결과값은 true 나 false로 반환,타입은 자료형이다고 이해하라

삼항연산자

조건문 if ~ eles의 sytantic sugar 변수 = 조건식 ? 참일때의 값 : 거짓일때의 값/조건식이 참일때

논리연산자

boolean 표현식에 대한 논리합OR(||) 또는 논리곱AND(&&) 또는 부정NOT(!) 연산을 수행하는 기호

변수의 자료형 알아보기

typeof 연산자를 이용하면 변수나 값으 자료형을 출력함

자료형 변환

함수와 연산자에 전달되는 값은 표현식에 적합하게 적절한 자료형으로 변환됨(묵시적 형변환)

하지만 상황에 따라 의도를 갖고 원하는 자료형으로 직접 변환해야 하는 경우도 있음 (명시적 형번환)

어떤 값을 숫자형으로 변환하고 싶다면 Number()사용

어떤 값을 문자형으로 변환하고 싶다면 string()사용

어떤 값을 블리언형형으로 변환하고 싶다면 boolean()사용
