scope

개요

변수가 사용될수 있는 범위를 scope라고 함

global scope: 선언된 변수는 어느 곳에서든지 접근 가능

local scope: 선언된 변수는 선언된 영역에서만 접근 가능

function scope:함수에 선언된 변수는 함수 안에서만 접근 가능

호이스팅hoisting

변수나 함수를 어디서 어떻게 선언하더라도 항상 컨텍스트내 최상위로 끌고 올라온뒤 undefined를 할당해두고 코드를 실행하는 것

var로 선언한 변수나 선언형 함수를 사용하는 경우 발생함

따라서 변수선언시 let 을 사용하고 화살표 함수선언을 이용하면 호이스팅 문제를 막을 수 있다
