자바스크립트 콜백 함수

개요

다른 함수의 인자로 사용되는 함수 또는 어떤 이벤트에 의해 호출되는 함수를 의미 함수를 값으로쓰는게 아니라 함수로 사용한다

사용하는 이유는 프로그램이 실행되는 동안 지속적으로 수행해야 하는 작업이 잇을때 사용-비동기 프로그래밍

어떤 정보를 원하는 쪽이 정보를 제공하는 쪽에게 반복적으로 확인 요구하는 것보다 정보를 제공하는 쪽이 정보를 원하는 쪽에게 통보하는 것이 효율적임

달걀을 삶을때 적정시간이 되었는지 확인해야 하거나 친구랑 놀고 잇을때 집에 가야할 시간을 확인해야 할때 타이머 콜백함수를 이용하는 것돠 동일

es5/Es6 에서는 promise 나 asycn/await 등이 추가 되어 콜백함수의 사용은 줄이는 추세임

타이머 함수

특정 시간이 지난후에 함수를 실행하려면 setTimeout함수를 사용 특정 시간마다 함수를 실행하려면 setInterval함수를 사용
