자바스크립트 반복문loop

개요

조건을 만족하는 동안 코드를 반복적으로 수행하는 문장

자바스크립트에서는 기본적으로 3가지 반복문(for, while, do-while)이 지원되고, 그외 for-loop, for-in과 ES6에서 추가된 for-of등이 있음

for

조건식이 거짓으로 판별될때가지(조건식이 참일 동안) 코드블록을 반복적으로 실행

for (초기식;판별식;증감식) { 반복구문 }

무한 반복문 -for

for문의 경우 초기식 판별식 증감식을 비워두면 반복을 끊임없이 실행하는 무한 반복문이 됨

중첩반복문 -for

2개 이상의 for 문을 겹쳐 사용하는 반복문 복잡ㅎ나 작업을 반족 실행할때 유용하게 사용

구구단 전체

5단가지

break

원하는 시점에 반복문이나 코드블록에서 벗어나기 위해 사용

continue

반복문의 코드실행을 현 지점에서 중단하고 반복문의 중감식이나 조건식으로 제어를 이동시킴 즉 반복을 일시적으로 건너뛰어 다음 반복으로 넘어감
